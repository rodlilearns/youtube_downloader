# youtube_downloader
Download mp3 and mp4 from Youtube.

## Pre-requisites
Need the pytubefix module in the working environment.
`pip3 install pytubefix`

Download mp3:
`python3 download_music.py`

## References:
* [pytubefix library](https://github.com/JuanBindez/pytubefix)
